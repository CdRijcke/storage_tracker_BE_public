STORAGE TRACKER
Hobby project on display. Backend code for a self-made 'smart' scale. Data is retrieved via a small scale that weighs
a food ingredient (oatflakes, peanutbutter, rice) and stores it into a database.
A front-end can retrieve the values to relief me from the doubt whether I need to buy another pack of rice
whilst being in the supermarket.
